# SpringBoot-Crawling-Project
스프링 부트 + 파이썬 파싱_영화 예매 사이트


## 🖥️ 프로젝트 소개
스프링 부트와 파이썬을 활용하여 크롤링한 데이터를 화면에 보여주는 웹 프로젝트 입니다.
<br>

### ⚙️ 개발 환경
- `Java 17`
- `JDK 1.8.0`
- **IDE** : STS 4.2
- **Framework** : Springboot(3.1.4)
- **Database** : MySQL 8.0
- **ORM** : Spring Data JPA
